## Logo Usage

The OrQuanta logo consists of the **OQ Monogram** (stylized O and Q) with a **quantum orbit ring**.

### Variants

| Variant | Usage |
|---------|-------|
| `full` | Icon + "OrQuanta" wordmark â€” use in headers, docs |
| `icon` | OQ monogram only â€” use in favicons, app icons |
| `wordmark` | Text only â€” use when icon is already shown nearby |

### Clear Space

Maintain minimum clear space equal to the height of the letter "O" in the wordmark on all sides.

### Don't

- Don't change the gradient colors
- Don't rotate the logo
- Don't add drop shadows or glow effects (built in)
- Don't place on busy backgrounds without a container
- Don't stretch or distort the aspect ratio
- Don't use the logo smaller than 24px icon size
